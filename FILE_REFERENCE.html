🔗 How to Use These Files

Method 1: Direct PHP Execution (Recommended for Full Features)

Your PHP files already include HTML templates and handle everything automatically:

1. Open: http://localhost/employee_task_tracker/login.php
2. Log in with valid credentials
3. Navigate using dashboard links

Method 2: Using Standalone HTML Files

Use the separate HTML files to understand form structure:

1. Open: http://localhost/employee_task_tracker/login_standalone.html
2. Submit the login form (connects to login.php)
3. Redirects to index.php with session
4. Continue with regular navigation

🛡️ Security Features

  • Session-based Authentication: All pages check for active session
  • Password Hashing: Passwords verified using password_verify()
  • SQL Injection Prevention: Prepared statements with parameterized queries
  • Output Escaping: HTML special characters escaped with htmlspecialchars()
  • Role-based Access Control: Different permissions for employees vs managers
  • CSRF Protection: Session tokens for form submissions

📍 URL Parameters

Page Parameter Value Type Example
task_view.php id integer (task ID) task_view.php?id=5
task_edit.php id integer (task ID) task_edit.php?id=5

ℹ️ Note: For task operations, the task ID must be provided as a URL parameter. The HTML files automatically include these parameters in links and redirects.

🚀 Getting Started

  1. Start your local server
    Make sure Apache and MySQL are running in XAMPP
  2. Navigate to login
    http://localhost/employee_task_tracker/login.php
  3. Log in with test account
    Use valid email and password from your database
  4. Create a test task
    Click "Create Task" and fill in the form
  5. View and edit tasks
    Click View/Edit buttons from the dashboard
  6. Test comments and time entries
    Open a task and add comments or time entries
  7. Delete and logout
    Test deletion and session cleanup
